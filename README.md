# Telecom-Churn-Model-EDA-Logistic-Regression-Business-Insights
Análisis completo del churn en telecomunicaciones usando Python. Incluye EDA, modelado con Regresión Logística, interpretación con SHAP y recomendaciones de negocio. Proyecto enfocado a identificar factores de abandono y mejorar estrategias de retención.
